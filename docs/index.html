Try in Dev Spaces Browser Extension

About

The Try in Dev Spaces Browser Extension adds a "Dev Spaces" button used to create cloud development environments in your Red Hat OpenShift Dev Spaces instance.

This extension can be configured to create development environments in your Dev Spaces or Eclipse Che® instance. By default, development environments are created on Eclipse Che® hosted by Red Hat running on the Red Hat Developer Sandbox.

The extension's source code is hosted on GitHub and is available under the MIT license. Pull requests are welcome. If you have any questions, concerns, issues or feature requests, please file an issue.

Privacy policy

The Try in Dev Spaces Browser Extension does not collect and send any data from your web browser to Red Hat.

  • Storage permissions are required to leverage the Chrome Storage API to allow your options to be saved locally and synced across devices.
  • Access to your data on github.com is required to determine the factory URL (i.e. the URL used to create a workpsace in a Dev Spaces instance) for the "Dev Spaces" button, and for adding the button onto the GitHub webpage.
  • Usage of the Red Hat Developer Sandbox is subject to the Red Hat Privacy Statement.

If you believe this privacy policy has been violated, please file an issue.
